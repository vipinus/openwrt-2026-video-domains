# openwrt-2026 — Chinese video sites FULL domain list (site + ancillary)
# Updated: 2026-04-18
#
# Purpose: exhaustive enumeration of every domain a typical CN video
# site needs to function — main host + API + CDN + image + analytics +
# login + player + ads-lite — so the list works even when dnsmasq's
# subdomain matching is defeated by third-party infra (e.g. iqiyi
# serving video segments from *.alicdn.com, bilibili from *.hdslb.com).
#
# Paired with direct.txt: direct.txt is the MINIMAL set (one anchor
# per site, relying on dnsmasq's *.domain matching); this file is the
# MAXIMAL set (every resolver-visible domain observed by a real player
# session). Use whichever fits the router's nftset strategy.
#
# Format: one domain per line; # for comments; dnsmasq matches subdomains.
